## 주요 발견

### 포지션별 최고 선수 (현역 기준)

### 포지션 크로스오버
- **하든·돈치치·SGA**: PG 등록이지만 실제로는 비전통 PG — Position Score는 낮을 수 있으나 능력치 자체는 절대 기준으로 평가
- **르브론·KD**: SF 등록이지만 PF급 체격·능력치 — SF Position Score에서 idef/dreb 가중치로 반영
- **야니스**: PF 등록 S+이나 실제로는 C·SF 겸용 — 포지션별 Score는 각 역할에서 다름

### F등급 집중 이유
전체 F등급 104명의 대부분은 2026 드래프트 프로스펙트 플레이스홀더 (능력치 30~44 범위). NBA 실제 데뷔 전 추정치이므로 낮은 Score가 정상.
