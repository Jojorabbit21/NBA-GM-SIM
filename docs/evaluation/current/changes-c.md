## 주요 패턴 분석

### 자주 발견된 오류 유형
1. **stl 과다 (C에게 65+)**: 마크 윌리엄스, 아이재아 잭슨, 폴 리드, 카만 말루아치, 딜런 카드웰, 크리스찬 콜로코 — 6명
2. **idef 과다 (OVR 대비)**: 아이재아 스튜어트(OVR82→idef87), 제이 허프(OVR77→idef80)
3. **패스 능력치 과다 (C에게 piq/pvis 85+)**: 도만타스 사보니스(piq92, pvis88)
4. **3점 능력치 비현실 (실제 3점 안 쏘는 선수)**: 재럿 앨런(c3 67)
5. **dreb/oreb/blk OVR 대비 과다**: 모리츠 와그너(dreb82), 마빈 베글리(oreb85), 드류 유뱅크스(blk81), 카를로 맛코비치(dreb81/blk80)

### 유지된 고수치 근거
- `루디 고베어 idef=97`: 4× DPOY, 역대급 수비 센터 → 유지
- `스티븐 아담스 oreb=90, str=92`: 포지션 특화 괴물 리바운더 → 유지
- `로버트 윌리엄스 III blk=88, idef=88`: 건강할 때 역대급 수비 임팩트 → 유지
- `요키치 pvis=99, piq=98`: 센터 역대 최고 패서 → 유지
- `웸반야마 idef=95, blk=95`: 현역 최고 블로커/수비수 → 유지
