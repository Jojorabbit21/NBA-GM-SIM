## 판단 근거 (수정하지 않은 주요 케이스)

- **르브론 siq=95, pvis=95**: 역대 최고 수준 패스마스터. 정당화 가능
- **KD mid=98**: KD의 signature 무기. 역대급 미드레인지 허용
- **KD siq=92, ocon=94**: KD 수준에서 정당화 가능
- **카와이 mid=96**: 카와이의 핵심 득점 무기. 허용
- **허버트 존스 blk=75**: 긴 팔 수비수 특성. SF치고 높지만 허용 범위
- **코리 키스퍼트 3c=88**: 코너 스팟업 전문가. 특화 스킬로 허용
- **비트 크레이치 3c=88, 3_45=86**: OVR 76이나 슈터 특화. 허용
