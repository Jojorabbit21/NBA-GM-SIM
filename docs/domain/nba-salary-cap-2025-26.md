## 2. 맥스 계약

경력(years of service) 기반으로 최대 연봉이 결정됨:

| 경력 | 캡 대비 % | 2025-26 1년차 맥스 |
|------|-----------|-------------------|
| 0~6년 | 25% | **$38,661,750** |
| 7~9년 | 30% | **$46,394,100** |
| 10+년 | 35% | **$54,126,450** |

### 계약 연수 & 인상률
- **자팀 재계약** (버드 라이츠): 최대 **5년**, 연 **8%** 인상
- **타팀 FA 계약**: 최대 **4년**, 연 **5%** 인상

### 슈퍼맥스 (지정 베테랑 계약)
- **조건**: 경력 7~8년 + 아래 중 하나 충족:
  - 직전 시즌 올-NBA 또는 DPOY 수상
  - 최근 3시즌 중 2시즌 올-NBA/DPOY 수상
  - 최근 3시즌 중 MVP 수상
- **효과**: 경력과 무관하게 **35% 맥스** (캡의 35%)
- **계약 기간**: 현재 계약 잔여 + 최대 6년
- **연 인상**: 8%
- **제한**: 현 소속팀만 제안 가능

### 데릭 로즈 룰
- 루키 스케일 선수가 3~4년차에 아래 조건 충족 시 **30% 맥스** (보통 25%):
  - 올-NBA 선정, 또는 MVP 수상, 또는 올스타 선발 2회

| 등급 | 조건 | 재계약 권한 | 최대 연수 | 인상률 |
|------|------|------------|----------|--------|
| **Full Bird** | 3+ 시즌 같은 팀 | **맥스까지** 캡 초과 재계약 가능 | 5년 | 8% |
| **Early Bird** | 2 시즌 같은 팀 | 이전 연봉의 **175%** 또는 리그 평균 연봉의 **105%** 중 큰 금액 | 4년 | 8% |

## 5. 럭셔리 택스 (2025-26 신규 세율)

2025-26부터 세율 체계가 변경됨. 첫 2구간은 인하, 나머지는 인상.

### 세율 테이블

| 구간 | 초과 범위 | 일반 세율 | 리피터 세율 |
|------|----------|----------|-----------|
| 1 | $0 ~ $5.685M | **1.00x** | **3.00x** |
| 2 | $5.685 ~ $11.370M | **1.25x** | **3.25x** |
| 3 | $11.370 ~ $17.055M | **3.50x** | **5.50x** |
| 4 | $17.055 ~ $22.740M | **4.75x** | **6.75x** |
| 5 | $22.740 ~ $28.425M | **5.25x** | **7.25x** |
| 6+ | +$5.685M마다 | +0.50x | +0.50x |

- **리피터**: 최근 4시즌 중 3시즌 이상 택스 납부 팀
- 세금은 시즌 종료 후 결산

### 계산 예시
총급여 $200M인 일반 납세 팀 (택스 라인 $187.895M, 초과분 $12.105M):
- 구간1: $5.685M × 1.00 = $5.685M
- 구간2: $5.685M × 1.25 = $7.106M
- 구간3: $0.735M × 3.50 = $2.573M
- **총 세금: ~$15.4M**

## 6. 에이프런 제한 (1차 / 2차)

### 6-1. 1차 에이프런 ($195.945M) — 하드캡 트리거

1차 에이프런은 그 자체로 하드캡이 아니지만, 아래 행위를 하면 **하드캡이 트리거**됨:
- Non-Taxpayer MLE 사용
- Bi-Annual Exception 사용
- Sign-and-Trade로 선수 영입

하드캡이 트리거되면 어떤 경우에도 1차 에이프런을 초과할 수 없음.

**1차 에이프런 초과 팀의 제한:**

| 항목 | 제한 내용 |
|------|----------|
| MLE | Non-Taxpayer MLE 사용 불가 → **Taxpayer MLE ($5.685M)만** 사용 가능 |
| Bi-Annual | 사용 **불가** |
| 트레이드 매칭 | 보내는 급여의 **100%**까지만 수령 가능 (초과 수령 불가) |
| 급여 합산 | 트레이드 시 복수 선수 급여 합산 **가능** |
| 바이아웃 | Non-Taxpayer MLE 이상 연봉의 바이아웃 선수 영입 **불가** |
| 캐시 트레이드 | **가능** |
| 이전 시즌 TPE | 사용 **불가** |

### 6-2. 2차 에이프런 ($207.824M) — 사실상 하드캡

**2차 에이프런 초과 팀의 추가 제한:**

| 항목 | 제한 내용 |
|------|----------|
| MLE | Taxpayer MLE **사용 불가** (최저연봉만 가능) |
| 트레이드 매칭 | 보내는 급여의 **100%**까지만 (1차와 동일) |
| **급여 합산** | 트레이드 시 복수 선수 급여 합산 **불가** (1:1만) |
| **캐시 트레이드** | **불가** |
| 바이아웃 | 바이아웃 선수 영입 **완전 불가** |
| TPE | 모든 TPE 사용 **불가** |
| **드래프트 픽** | 7년 후 1라운드 픽 트레이드 **불가** (6년까지만) |
| **리피터 페널티** | 4시즌 중 2시즌 2차 에이프런 → 7년 후 1라운드 픽이 **자동으로 30번으로 이동** |

### 에이프런별 트레이드 매칭 요약

| 팀 상태 | 매칭 룰 | 급여 합산 | 캐시 |
|---------|---------|----------|------|
| **캡 미만** | 잔여 캡 스페이스 내 자유 + $250K | — | ✓ |
| **캡 초과 ~ 1차 에이프런 미만** | 보내는 급여에 따라 단계별* | ✓ | ✓ |
| **1차 에이프런 이상** | **100%** (초과 수령 불가) | ✓ | ✓ |
| **2차 에이프런 이상** | **100%** + **합산 불가** | ✗ | ✗ |

*캡 초과 ~ 1차 에이프런 미만 팀의 단계별 매칭:
- 보내는 급여 ~$7.5M 이하: **200% + $250K**
- 보내는 급여 $7.5M ~ $29M: **보내는 급여 + $7.5M**
- 보내는 급여 $29M 초과: **125% + $250K**

## 8. 방출 / 웨이버 / 바이아웃

### 8-1. 웨이버 (Waiver)
- 선수를 방출하면 48시간 웨이버 기간
- 다른 팀이 클레임 가능 (남은 계약 인수)
- 클레임 없으면 FA 전환

### 8-2. 보장 계약 방출 시 캡 히트
- 보장 계약 선수 방출 → 남은 **보장 급여 전액이 데드 머니**로 캡에 잔류
- 즉, $40M/3년 보장 계약 방출 시 → **$40M × 3년 = $120M 전액** 팀 부담 (줄이는 것은 불가)
- 단, 바이아웃 협상으로 금액 감면 가능

### 8-3. 바이아웃 (Buyout)
- 팀과 선수가 합의하여 보장 금액을 **할인**하고 방출
- 바이아웃된 선수는 FA로서 다른 팀과 자유 계약 가능
- 새 계약 급여의 일부가 원래 팀의 부담에서 **상계(set-off)** 가능

### 8-4. 스트레치 프로비전 (Stretch Provision)
- 방출 선수의 남은 보장 급여를 더 긴 기간에 걸쳐 분산:
  - **분산 기간 = (잔여 계약 연수 × 2) + 1년**
  - 예: 3년 $90M 남은 선수 → 7년에 걸쳐 $12.86M/년 캡 히트
- 단기 캡 부담을 줄이지만, 장기간 데드 머니가 남음
- **제한**: 한 시즌에 팀이 스트레치한 총 금액이 캡의 15% 초과 불가

### 8-5. 상계 (Set-off)
- 방출된 보장 계약 선수가 새 팀과 계약하면, 원래 팀의 부담이 일부 감소
- 감소분 = (새 계약 급여 - 최저연봉 비례분) ÷ 2

## 9. 기타 규정

### 노 트레이드 조항 (NTC)
- 경력이 긴 고연봉 선수가 보유 가능
- 선수 동의 없이 트레이드 불가

### 팀 최저 급여 (Salary Floor)
- $139.182M (캡의 90%)
- 미달 시 차액을 선수들에게 분배

### 트레이드 제한 기간
- 새로 계약한 선수: **3개월** 또는 시즌 시작 후 **30일** 중 나중 시점까지 트레이드 불가
- 12월 이후 트레이드된 선수: 해당 시즌 트레이드 데드라인에 **재합산(re-aggregation)** 불가

### 캡 홀드 (Cap Hold)
- FA 협상 중인 선수에 대해 예상 계약금이 캡에 잡혀 있음
- 캡 스페이스를 확보하려면 해당 선수의 버드 라이츠를 포기해야 함
